New Page Link plugin converts all your external or internal links in the body to always open in a new tab.

== Description ==

New Page Link plugin makes all your hyperlinks in your body area to open in new tabs.

Installation Guide

[youtube https://youtu.be/73H4JuZQV40]

Major features of New Page Link include:

*   Changing the behaviour of link or links to open in new tabs or in the same window on clicking
*   An options page to control open link in new page functionality

1. You get a lot more flexibility in the pro version as you can make unlimited or limited number of links to open in a new tab on clicking so [Go Pro](https://metalogics.tech/product/new-page-link/)
2. Please also donate to help me buy a cup of coffee and rate me good to encourage my work [Donate](https://metalogics.tech/donate/).


Below are the steps to install New Page Link plugin

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the settings menu in the dashboard and then look for new link menu in it (settings/newlink).
4. Click on new link menu 
5. Type the value 'Yes' to make links in body of your pages to open in new window
6. Type 'No' or anything else to keep them opening in the same window
7. And you are done !

Screenshots

Click screenshot tab to see the options area under settings and then clicking on newlink option. There you have to type Yes to make it open everything in body of the page to open in new tab. Change it to No or anything else to open links in the same window.
